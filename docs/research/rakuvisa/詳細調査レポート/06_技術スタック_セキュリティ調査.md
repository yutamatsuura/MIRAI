**調査日:** 2026年1月15日

## エグゼクティブサマリー

RakuVisaは、日本の在留資格（ビザ）申請をオンラインで完結できるクラウドベースのプラットフォームです。入管庁とのAPI連携により、外国人労働者、雇用企業、登録支援機関、行政書士・弁護士が協働してビザ申請業務を行うことができます。

## 15. システムアーキテクチャ（推測）

### 15.1 システム構成

```
[外国人ユーザー] ─┐
[雇用企業]       ─┤
[登録支援機関]   ─┼─→ [RakuVisaプラットフォーム] ←→ [入管庁API]
[行政書士]       ─┤      ├─ バックエンド（C#/Java）
                  │      ├─ フロントエンド（React/TypeScript）
                  │      ├─ データベース（PostgreSQL/Oracle）
                  └─→    └─ クラウドストレージ（書類・録画）
```

### 15.2 データフロー

1. **申請データ作成:** 各関係者が必要情報を入力
2. **書類自動生成:** システムが申請書類を自動作成
3. **行政書士確認:** 専門家が内容確認・電子署名
4. **入管庁送信:** API経由で電子申請
5. **進捗追跡:** リアルタイムで状況確認
6. **結果通知:** 結果の自動通知・記録

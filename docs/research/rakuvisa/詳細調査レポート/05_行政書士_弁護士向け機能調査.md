# Rakuvisaプラットフォーム 行政書士・弁護士向け機能 完全調査レポート

**調査日**: 2026年1月15日
**調査対象**: RAKUVISA株式会社が提供する在留資格申請オンラインプラットフォーム「RakuVisa」

---

## エグゼクティブサマリー

RakuVisaは、日本で初めて出入国在留管理庁とAPI連携した在留資格申請のオンラインプラットフォームです。行政書士・弁護士は、このシステムを活用することで、顧客企業・外国人の在留資格申請業務を効率化し、法令遵守を徹底しながら業務品質を向上させることができます。

**主な特徴:**
- 入管庁とのAPI直接連携による完全オンライン申請
- 多者間協働プラットフォーム（企業・外国人・行政書士）
- AI予測による審査期間の可視化
- 専門家向けの電子署名・承認ワークフロー
- 初期費用・月額費用無料（申請毎の従量課金）

## 1. 顧客（企業・外国人）の一元管理

### 1.1 多者間協働管理システム

**機能概要:**
- 申請人（外国人）、所属機関（雇用企業）、登録支援機関、行政書士の4者がオンラインで連携
- リアルタイムで進捗状況を全関係者が同時に把握可能
- 業務分担と責任範囲の明確化

**行政書士のメリット:**
- 複数の企業顧客・案件を一元管理
- 顧客企業ごとの申請状況をダッシュボードで俯瞰
- 登録支援機関との協働業務を効率化

### 1.2 クラウドベース顧客データ管理

**機能:**
- 申請データのクラウド保存・履歴管理
- 過去の申請履歴の検索・参照
- 顧客情報の長期保存とセキュアな管理

**特徴:**
- 入管への申請後も全データがクラウドに保存
- 監査対応時の過去データ即時取り出し
- 特定技能の定期面談記録は契約終了日から1年以上保管（法令対応）

### 1.3 複数顧客の効率的管理

**対応範囲:**
- 登録支援機関向けプラン（RakuVisa for TSK）による大量案件管理
- 企業向けプラン（RakuVisa for Biz）による企業単位管理
- 行政書士が複数の顧客企業を横断管理

## 2. 在留資格申請書類の作成支援

### 2.1 対応在留資格

**ほぼ全ての在留資格に対応:**
- 就労系: 技術・人文知識・国際業務、特定技能1号・2号、経営・管理、介護、特定活動など
- 身分系: 日本人の配偶者等、家族滞在など
- 申請種類: 認定、変更、更新（短期滞在と永住を除く）

### 2.2 オンライン書類作成機能

**主要機能:**
- 雇用契約書のオンライン作成
- 条件書（労働条件通知書）のオンライン作成
- 支援計画書のオンライン作成
- 全ての必須書類をブラウザ上で完結

**データ入力支援:**
- 入力項目のガイド表示
- 自動保存機能（従来の入管オンラインシステムにはない機能）
- 入力内容の一時保存により、データ消失リスクを排除

### 2.3 多言語翻訳機能

**特定技能1号の自動多言語対応:**
- 翻訳必須な全ての書類について自動多言語翻訳
- 外国人本人が母国語で内容を確認可能
- 翻訳作業の大幅な効率化

**対応言語:**
- 特定技能対象国の言語に対応
- システムによる自動翻訳で、手作業の翻訳工数を削減

### 2.4 テンプレート機能

**確認項目のテンプレート化:**
- 行政書士が確認すべき項目をテンプレート化
- 申請種類・在留資格ごとの標準チェックリスト
- 確認漏れの防止と業務品質の標準化

**自動アラート:**

## 11. 電子署名・承認ワークフロー

### 11.1 行政書士による電子署名

**法令遵守の仕組み:**
- 行政書士が申請内容を確認
- 電子署名を付与して申請
- 入管法・行政書士法に完全準拠

**マイナンバーカード活用:**
- 行政書士のマイナンバーカードを使用
- スマートフォンまたはカードリーダーで読み取り
- 公的個人認証による厳格な本人確認

### 11.2 確認・承認フロー

**標準ワークフロー:**
1. 外国人本人または企業が基本情報を入力
2. 必要書類をアップロード
3. 行政書士がテンプレート化された確認項目をチェック
4. 内容に問題がなければ電子署名
5. 入管庁へオンライン申請

### 11.3 申請取次の法的要件

**資格要件:**
- 出入国在留管理庁に申請取次者として届出済み
- 有効な証明書を保有する行政書士・弁護士

**登録要件:**
- 行政書士事務所、行政書士法人、弁護士事務所、弁護士法人
- 登録番号、事務所情報の提供
- 行政書士法及び関係法令の遵守への同意
